В центре науки

Проект направлен на популяризацию науки, формирование сообщества молодых ученых и педагогических работников в системе образования г. Москвы

Целевая аудитория: ученики, учителя, учёные.

Задача: разработать для квиза учёных и учителей карточку промопоста, бэйджи и сертификаты для участников

Решение: сделать бэйджи с разными цветами, исходя из целевой аудитории, добавить динамики в макеты, соответствующей атмосфере квиза

Задача: к научным курсам для школьников разработать афишу для школ, добавить маскота, сделать карточку для промопоста с фото, создать уникальные стикеры для научных фестивалей.

Решение: маскот дает дополнительный акцент на целевую аудиторию мероприятия, афиша отражает дизайн веб-страницы ивента. Стикеры соответствуют тематике научных фестивалей — химия, социо-гуманитарные науки, программирование. Цитата Аристотеля отсылает к истории научного знания, единства биологического и социального в человеке.

Задача: сделать благодарственные письма для школ в официальном стиле.

Решение: использовать минимализм и цветовую палитру по предметным областям аналогично личным сертификатов участников, форма подчеркивает тональность содержания — выражение благодарности за образовательные программы, раскрывающие потенциал каждого ученика.
